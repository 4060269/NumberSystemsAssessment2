Bits and Bytes

At the core, computer systems are made up out of bits. Bits is shorthand for Binary Digits.

These are 0's and 1's.

As a standard, these can be organised into groups of 8, which are referred to as bytes.

Below, enter individual bits, which will be converted into a decimal number. For instance 10000000 will output 128, because the 1 is in the 128th position.

Student Note: Store the individual bits (0 or 1) into an array (e.g. bits). Then you can run the command: let num = +bits.join(""), where bits is the name of the array.
The command converts the array of integers into an integer.
With the example above the array 1,0,0,0,0,0,0,0 would be converted into the integer: 10000000. Then this number can be converted from binary into decimal. This can be researched.
