Sum and Product

Welcome!

This page will generate a given number of random numbers, and output the sum and product of those numbers.

Press the button below, enter the number of random numbers you wish to generate and then see the answer.

Note to students: Store the running sum and product into a dictionary called numberDetails. For instance the dictionary might look like: 

        
            numberDetails = {
                sum: 0;
                product: 1;
            }
